#Image Magick 
baixar o ImageMagick-7.0.8-39-Q16-x64-dll.exe no site https://imagemagick.org/script/download.php#windows

Nos passos da instalação, marcar a opção
Install legacy utilities (e.g. convert)
(esse é o pulo do gato pra versão 7)

abra o cmd (se estiver aberto, fecha e abre novamente)
digite:
magick identify --version

Se aparecer a versão 7, então deu certo =)
obs.: caso esteja usando o terminal via vs code, reinicie o vs code
